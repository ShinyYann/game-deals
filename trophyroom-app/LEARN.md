## ✍️ 入门心得（我的建议）

### 1. 不要追求"理解了再动手"
编程不是数学。**先照抄，再理解。** 你把 TrophyRoom 的代码改一个颜色、改一个按钮文字，哪怕完全不懂原理，也是在进步。

### 2. 遇到 Bug 先读错误
Flutter 的错误提示是中文的。不要一红就慌，**读第一行错误文字**，80% 能看出哪错了。

### 3. 每次只改一样东西
不要一次改 10 行代码然后跑不动了不知道哪行改坏了。**改一行→跑一次→确认没问题→再改下一行。**

### 4. 复制粘贴不丢人
90% 的代码都是复制粘贴改一改。全世界的程序员都这么干。

### 5. AI 是你的老师
你遇到不会的，直接问我。**但不要让我替你写**——先自己试试，试不通再来问我，学得最快。

## 🔧 你能做的 5 个入门练习

按顺序，从简单到难：

### 练习 1：改标题文字
```
找到 main.dart 里的 "🏆 TrophyRoom"
改成 "🏆 Yann 的游戏屋"
然后在手机上看到效果
```

### 练习 2：改主题色
```
找到 Color(0xFFA855F7)
这个紫色的值，改成 0xFF00FF00（绿色）
看整个 APP 颜色变没变
```

### 练习 3：加一个折扣游戏
```
找到 _offlineGames 这个列表
照葫芦画瓢加一条：
{"name": "我的新游戏", "discount": "99%", "price": "HK\$1.00", "original": "HK\$100.00", "platform": "PSN"}
```

### 练习 4：改底部导航的文字
```
找到 BottomNavigationBarItem
把 "首页" 改成 "主页"，"折扣" 改成 "特价"
```

### 练习 5：加一个 Tab（进阶）
```
在 BottomNavigationBar 里加第三个 item
对应的页面显示 "正在开发中"
```

## 📁 TrophyRoom 项目结构（一句话版）

```
trophyroom-app/
├── lib/
│   └── main.dart          ← 90% 的代码都在这里（你主要改这个）
├── android/               ← 安卓相关（不用动）
├── pubspec.yaml           ← 依赖配置（偶尔动）
└── test/                  ← 测试（暂时不管）
```
