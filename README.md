# 🏆 TrophyRoom / 奖杯屋

> *"你说的这些我都要，目标就是最强"* — Yann, 2026-05-01 04:07

Yann 的私人全平台游戏多功能站。自用不推广，邀请好友各自看各自的数据。

## 起源

2026-05-01 凌晨，Yann 和 King 碰撞了一整晚，确定了 TrophyRoom 的灵魂：

> 🎨 **"我对特效需求很高，专业游戏特效级是我的期望"**
> → 因此有了 Phase 1-4 特效升级路线图

> 👤 **"我要不同人使用时登录不同账号，我要和我的朋友们使用"**
> → 因此有了多账号系统，每人独立数据

> 🐉 **"我还是一个资深宝可梦玩家，我想为个别游戏定制专属攻略站"**
> → 因此有了宝可梦殿堂 + 更多专属攻略站

> 💡 **"我希望你能不断蹦出想法，我们可以通过不断的碰撞擦出新的火花"**
> → TrophyRoom 永不完工，持续进化

## 访问

[https://shinyyann.github.io/game-deals](https://shinyyann.github.io/game-deals)

## 功能

- 🏆 奖杯查询（PSN 奖杯/Steam 成就）
- 🎯 多平台折扣聚合（PSN 港服 / Steam 国服 / Switch 港服）
- 💸 PSNine 新史低追踪
- 💬 P9 社区情报（入库会免 / 新史低 / 白金攻略）
- 🎮 Mod 资讯
- 📰 游戏快讯
- ⏰ 每日自动更新

## 技术细节

- Python 脚本抓取各平台折扣数据
- GitHub Actions 每天自动运行 (08:00 北京时)
- 生成静态 HTML 部署到 GitHub Pages
- PWA 支持，可添加到主屏幕
- APK 自动打包（GitHub Actions → Release）

## 🎮 特效升级路线图 — VFX Roadmap

> 目标：专业游戏级别 UI 特效，不做花架子，每一个特效都有"哇塞"感

### Phase 1 — 视觉冲击（当前重点）
| # | 特效 | 描述 | 状态 |
|---|------|------|------|
| 1 | 🃏 **卡片 3D 悬浮** | 鼠标划过游戏卡片像 UI 面板一样 3D 倾斜，跟随视角 | ⏳ |
| 2 | 🌟 **鼠标跟随光晕** | 整个界面有动态光斑/射线跟随鼠标位置移动 | ⏳ |
| 3 | ✨ **粒子系统升级** | 从简单文字飘动升级为能量粒子/星尘/星云系统 | ⏳ |
| 4 | 📦 **卡片入场序列** | 数据加载时卡片逐个弹出，像打牌一样有节奏登场 | ⏳ |
| 5 | 🌈 **Tab 转场特效** | 类似《命运2》的棱镜翻页/片状过渡，非简单淡入 | ⏳ |

### Phase 2 — 材质与氛围
| # | 特效 | 描述 | 状态 |
|---|------|------|------|
| 6 | 💎 **玻璃态 UI 面板** | 磨砂玻璃 backdrop-filter，毛玻璃卡片面板 | ⏳ |
| 7 | 🌊 **流体渐变背景** | 背景像液态金属/极光般缓慢流动，Tab 切换变色 | ⏳ |
| 8 | 🔥 **霓虹发光边框** | 游戏 UI 风格的扫描线发光边框，动态闪烁 | ⏳ |
| 9 | 🌓 **动态光影** | 界面光照随时间/操作变化 | ⏳ |

### Phase 3 — 交互与反馈
| # | 特效 | 描述 | 状态 |
|---|------|------|------|
| 10 | 🔊 **声效反馈** | 点击/切换/弹窗的轻量音效反馈 | ⏳ |
| 11 | 🌀 **涡旋加载动画** | 数据加载时游戏风格能量漩涡，非转圈 | ⏳ |
| 12 | 💥 **弹窗入场特效** | 弹窗不再是简单弹出，粒子爆散/能量冲击式入场 | ⏳ |
| 13 | 📊 **数据动效** | 价格数字跳动、进度条流光、星星闪耀 | ⏳ |

### Phase 4 — 持续打磨
- 根据 Yann 实际体验反馈持续优化
- 关注性能，确保手机/低配设备流畅
- 不堆砌特效，每个效果有意义、有节奏
